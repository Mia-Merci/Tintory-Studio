Graphic Fashion Culture

Tintory 将图像视为一种历史材料。字体、衣料、银饰、秀场、纹样、版面与空间，并不是彼此孤立的视觉事件，而是同一套观看方式在不同媒介中的显影。

Ⅰ / Manifesto

三个领域，一条线索

平面设计是语法的底层——字体、配色、构成，是视觉思维的元语言。时尚观察是语法在身体上的投射——廓形是空间构成，面料是质感对比，秀场是沉浸式排版。

文化探索是语法的源头——为什么苗族银饰用黑白对比？为什么日式侘寂偏爱不对称？每一个民族的审美选择，都是一套完整的视觉方法论。

方法而非风格

我们不追求统一的 "Tintory 风格"。我们追求一套可迁移的视觉分析方法——无论是一张海报、一场秀、还是一个民族的服饰体系，我们问的是同一个问题：它的美来自什么结构？

这种追问使我们的产出不局限于某一类媒介。平面设计师、时尚评论人、文化研究者——在我们的工作方式中，这些身份是同一种能力的三种投射。

Ⅱ / Disciplines
Graphic

建立视觉语言的底层秩序。我们关注字体、网格、信息层级、色彩比例与印刷质感，让设计成为一种可被分析、延展与迁移的结构。

Poster / Identity / Editorial / UI System
Fashion

将服装视作身体上的建筑。廓形、面料、秀场音乐、空间装置与品牌叙事，共同构成一场关于时代气质的视觉文本。

Runway / Brand Archive / Styling / Review
Culture

进入民族美学、地域纹样、工艺系统与历史现场，在被忽略的文化材料中，重新发现颜色、比例、符号与空间的秩序。

Ⅲ / Studio
视觉身份

为品牌、独立项目与文化机构建立视觉识别系统，包括 Logo、字体策略、色彩系统、版式规范与应用延展。

编辑设计

为杂志、展览、出版物、专题页面与数字内容建立编辑节奏，让文字、图像与留白形成清晰而有气质的阅读关系。

视觉研究

围绕时尚、文化、地域美学与图像系统进行研究写作，输出文章、策划提案、品牌分析与视觉方法论。

创意方向

为影像、摄影、社交媒体内容与品牌项目提供概念设定、视觉语气、拍摄方向、叙事结构与整体美术把控。
